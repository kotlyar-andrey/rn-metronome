# rn-metronome

## Installation

```sh
npm install rn-metronome
```

## Usage

### Metronome:

```js
import { Metronome } from 'rn-metronome';

onPress = () => {
  Metronome.play(100); // play with bpm=100
};

// ...

onPress = () => {
  Metronome.stop(); // stop playing
};
```
